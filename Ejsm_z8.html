Wewnętrzny arkusz stylów

  1. Gdzie jest wstawiany

    2. Wstawia się zawsze w części nagłówkowej dokumentu (pomiędzy znacznikami head).

  2. Składnia teoretyczna

    3. <head>
       <style type="text/css">
         selektor1 { cecha: wartość; cecha2: wartość2; }
         selektor2 { cecha: wartość; cecha2: wartość2; }
       </style>
    </head>

  3. Jakie mogą być selektory

    4. Selektorem może być praktycznie dowolny znacznik, np. p (akapit), h1 (tytuł), td (komórka tabeli) i inne.

  4. Jakie mogą być cechy i wartości

    5. Jako "cecha"(w deklaracji stylu -powyżej) należy wpisać o jakie konkretnie atrybuty formatowania nam chodzi (opisane w kolejnych rozdziałach).Natomiast jako wyraz "wartość" wpisujemy dokładną wartość atrybutu. Zwróć uwagę, że jednemu selektorowi możemy nadać kilka atrybutów (cech). Są one wtedy rozdzielone średnikami.
